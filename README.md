# 使用方式

1. 你的电脑需要安装了`python3.6`或以上
2. 将全部文件解压到 `plugins/SlimeCustomizer`
3. 运行`main.bat`或`main.sh`，程序将会生成`translated`文件夹
4. 将`translated`直接放到`RykenSlimeCustomizer/addons`内即可
